# Formation SEI Accuracy Stability Retention Simplicity

Field manual — read the scores honestly

The index puts 35% of its weight on not changing the data (20% retention + 15% simplicity). The base model here is a Random Forest, which ignores feature scaling — so on clean datasets the minimal formations (reconnaissance, and its kin) win almost by construction, and the aggressive ones lose. That is a real property of this metric-and-model pairing, surfaced rather than hidden.

Where SOFIDR earns its keep is matched terrain: on the imbalanced battlefield, guerrilla deploys SMOTE and edges out doing nothing. Switch the base model to logistic regression in the live app and the formations spread apart.

This page shows captured output. The live version computes every score on demand from your own CSV via the FastAPI engine — deploy it with vercel deploy --prod.
